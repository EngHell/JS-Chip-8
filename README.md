# JS-Chip-8
Chip-8 emulator written in javascript, can load localfiles direct to the browser memory using the FILE api

## Usage
1. Unzip chip8-js.zip ([Download](#download))
2. Open index.html.
3. Chose a rom file
4. Click load rom

## Keypad bindings
```
Keypad                   Keyboard
+-+-+-+-+                +-+-+-+-+
|1|2|3|C|                |1|2|3|4|
+-+-+-+-+                +-+-+-+-+
|4|5|6|D|                |Q|W|E|R|
+-+-+-+-+       =>       +-+-+-+-+
|7|8|9|E|                |A|S|D|F|
+-+-+-+-+                +-+-+-+-+
|A|0|B|F|                |Z|X|C|V|
+-+-+-+-+                +-+-+-+-+
```
